Conclusiones

La computación de alto rendimiento (HPC – High Performance Computing), promueve el modelo de programación paralela, poder usar múltiples procesadores simultáneamente para resolver un problema computacional complejo. Las plataformas de cómputo que soportan la computación de alto rendimiento responden a las necesidades de aplicaciones que requieren de ejecución de muchas tareas y análisis de gran volumen de datos, lo que conlleva a la exigencia de gran poder de cómputo para ser ejecutadas en un tiempo razonable. La programación paralela involucra muchos aspectos que no se presentan en la programación secuencial. El diseño de programas paralelos tiene que considerar entre otras cosas, el tipo de arquitectura sobre la cual se va a ejecutar el programa, las necesidades de tiempo y espacio que requiere la aplicación, el modelo de programación paralelo adecuado para implantar la aplicación y la forma de coordinar y comunicar a diferentes procesadores para que resuelvan un problema común.
